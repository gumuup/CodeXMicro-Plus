# 更新日志

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## 2.0.1 — 2026-07-19

- 将面板合成的快捷键和文本事件直接投递到当前 Codex 进程，修复新版 Codex 中 FAST 等操作无响应的问题。
- 安装包沿用稳定的代码签名要求，减少应用更新后重复申请辅助功能权限。
- 版本号与通用 DMG 安装包更新为 2.0.1。

## 2.0.0 — 2026-07-19

### 全局快捷键

- 为 A1–A6、四向摇杆、FAST、同意、拒绝、新任务、标注切换、语音、Codex 状态以及推理强度增减提供全局快捷键。
- 支持直接右键面板按键或旋钮半区录制、改绑和清除快捷键，并持久化用户配置。
- 增加快捷键冲突检测、重复绑定迁移、录制超时与取消反馈，避免合成按键事件反向触发面板动作。

### 交互与分发

- 快捷键录制时临时激活悬浮面板，完成后恢复原先的前台应用。
- 扩展面板四边拖动区域，更新侧边版本标识，并优化辅助功能授权提示行为。
- 未配置 Developer ID 时，DMG 改用稳定的本地开发签名，减少升级后重复申请辅助功能权限。
- 延续 1.1.3 的悬浮位置设置与关闭后重新打开修复。

## 1.1.3 — 2026-07-18

- 在通用设置中新增“悬浮位置”，支持持久化选择置顶模式或沉底模式，并即时调整面板层级。
- 修复点击右上角“×”隐藏面板后，再次打开应用无法恢复面板的问题。

## 1.1.2 — 2026-07-18

- 修复任务标题较长、SQLite 查询结果超过管道缓冲区后，任务状态和 Token 用量刷新永久卡住的问题。
- 修复 GPT-5.3-Codex-Spark 独立额度错误覆盖 Codex 主额度，导致本周剩余比例显示不准确的问题。
- 增加大输出进程读取和混合额度日志的回归测试。

## 1.1.1 — 2026-07-17

- 将仓库说明、macOS 应用与安装包的用户可见名称统一为 `CodeXMicro++`。
- 应用和安装包文件名改为 `CodeXMicro++.app` 与 `CodeXMicro++-1.1.1-universal.dmg`。
- 删除实体控制器插件、场景、管理工具及相关 Node.js 代码，仓库仅保留 macOS App。
- 保留 Bundle ID、签名目录与窗口状态键等技术标识，确保升级兼容。

## 1.1.0 — 2026-07-17

首个公开开源版本。

### macOS 虚拟控制面板

- 原生 SwiftUI 4 × 4 悬浮面板，支持移动、等比例缩放、跨桌面与始终置顶。
- 最近 6 个 Codex 任务的实时状态与一键跳转。
- 四向任务摇杆、四档推理旋钮、Fast、同意、拒绝、新任务和按住说话。
- CodeX 状态键显示本周剩余 Token 与累计消耗，约每 20 秒刷新。
- 可搜索的分类工具箱，覆盖任务、代码、Git、导航、界面与常用工作流。
- Force Touch 触觉强度、机械按键音与辅助功能权限设置。

### 分发

- Apple Silicon / Intel 通用 DMG。
- SHA-256 校验文件。
- MIT License 与完整源码。
