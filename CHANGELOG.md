# 更新日志

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## 3.2.0 — 2026-07-23

### Ulanzi Dial 与复合控制器

- 原始 HID 监听扩展到鼠标、摇杆、游戏手柄、键盘、数字键盘、多轴控制器和 Consumer Control 顶层集合，适配 Ulanzi Dial 及常见复合控制器。
- 支持 Generic Desktop Dial / Wheel、Consumer 媒体键、Button 页面按键及厂商自定义控件；Ulanzi Dial 顺时针和逆时针可保存为两个独立映射。
- 相对旋钮和滚轮的每个刻度会转换为完整的按下/松开脉冲，并将单次事件的脉冲数限制在 16，避免异常设备值造成操作风暴。
- 厂商自定义页面只接收相对控件或逻辑范围为 0–1 的二值按键；过滤绝对轴和状态字段，避免误录以及快捷动作停留在按下状态。
- HID 标识新增用途页、方向和设备顶层用途，匹配不依赖可能变化的设备显示名称；普通鼠标主键继续保留安全录制限制。
- 播放/暂停、上一首、下一首、静音、音量和旋钮方向新增可读显示名称。

### 兼容性与验证

- HID 标识采用自定义解码并为新增字段提供默认值，3.1.0 及更早版本保存的宏键映射可无损读取。
- 增加旧版 JSON、鼠标主键安全、Ulanzi Dial 双向匹配及 Consumer 播放/暂停标注测试。
- 应用版本、构建号、Codex 客户端标识、安装说明和通用 DMG 文件名统一更新为 3.2.0（构建 320）。

## 3.1.0 — 2026-07-22

### 鼠标与 HID 按键监听

- 自定义按键监听扩展到鼠标左键、右键、中键与侧键，支持普通鼠标按键或搭配 Control、Option、Shift、Command 的组合映射。
- 新增原始 IOKit HID 鼠标按钮监听，可识别部分不会转换成常规 macOS 鼠标事件的独立宏键，并按厂商、产品和用途编号持久化绑定。
- 鼠标与 HID 监听补齐按下、重复抑制、松开及重新注册时的释放语义，保持语音等按住型功能行为一致。
- 快速启动、轮盘及全部面板功能均可使用鼠标按键；轮盘单项直达操作支持原始 HID 宏键。
- 禁止无修饰键的鼠标左键或右键被完全接管，避免用户失去基础点击能力。
- 键盘快捷操作拒绝误用鼠标绑定，系统快捷键冲突检测也会正确跳过鼠标/HID 类型。

### 快速启动与可靠性

- 快速启动显示悬浮面板时改为定位到当前光标附近，并在多显示器和屏幕边缘自动限制到可用区域。
- 修复只配置原始 HID 宏键，或键盘事件监听不可用时，HID 映射可能被清理而无法触发的问题。
- 增加鼠标绑定、HID 标识、旧配置兼容、按下/松开匹配及光标定位边界测试。

### 轮盘编辑与图标

- 每个全局或应用轮盘扩展为 10 组可即时切换、自动保存的预设组合；旧版单组配置会无损迁移到第 1 组，其余组合提供 8 个空操作位。
- 新增轮盘操作复制与粘贴，支持列表/预览右键菜单及非文字编辑状态下的 `⌘C`、`⌘V`，可跨位置和预设组合复用完整动作。
- 图标库新增 Emoji、扩展 SF Symbols 与本地图片三类来源，并按人物、自然、办公、开发、设计、媒体等目录组织。
- 图标搜索支持中文、英文、拼音片段、子序列和轻微拼写错误；本地图片导入后保存在 Application Support 并可跨预设复用。
- 轮盘、预览、列表和编辑器统一渲染系统图标、Emoji、本地图像及真实应用图标。

### 数据与系统集成

- 新增“数据”设置页，可选择数据保存目录、从目录恢复或覆盖、创建/恢复 ZIP 备份，并检查或修复设置和自定义图标引用。
- 支持重建应用数据及检查、清理未被任何轮盘预设引用的自定义素材；相关危险操作均要求二次确认，且不会修改 `~/.codex`。
- 应用启动前会从所选数据目录载入配置，并在设置变化和退出时原子保存当前偏好。
- 通用设置新增 macOS 登录项管理，可一键开启或关闭开机自启动，并在需要系统审批时跳转到登录项设置。

### 构建与分发

- 应用版本、构建号、Codex 客户端标识、安装说明和通用 DMG 文件名统一更新为 3.1.0（构建 310）。

## 3.0.0 — 2026-07-21

### 快速启动轮盘

- 在“快速启动”右侧新增独立的“轮盘”设置页；原有 `⌥Space` 快速启动保持不变，轮盘默认使用 `⌃Z`。
- 轮盘在当前光标旁显示，支持按住、移向扇区、松手执行；轻按保持显示，也可点击操作或再次按键关闭。
- 支持添加 Codex 工具箱全部预设、任意快捷键、应用程序、插件入口、网址、粘贴文本、文件夹和 macOS 快捷指令，配置会保存在本机。
- 新增可排序的轮盘操作编辑器、实时预览和可搜索图标库，包含全部 Codex 工具箱图标及百余个常用 SF Symbols。
- 轮盘支持按应用保存独立预设，并为每个轮盘项录制直达快捷键；组合键按当前前台应用匹配，支持清除、重复改派和系统占用提示。
- 新添加的应用预设改为 8 个“未设置”空位，不再复制 ChatGPT 的 Codex 操作；用户可逐项选择类型、功能、图标与快捷键。
- 新增可持久化的轮盘“全局模式”；开启后所有应用统一使用全局轮盘及其单项直达快捷键，关闭后恢复按应用匹配，既有预设不会被覆盖。
- 全新安装默认开启轮盘全局模式，并内置当前推荐的 8 项全局应用轮盘与 7 项 ChatGPT 专属轮盘；升级用户继续保留既有设置。
- 轮盘操作类型新增“系统应用”，自动读取 macOS 的系统应用与实用工具，可直接选择系统设置、App Store、活动监视器、终端和磁盘工具等应用。
- 快速启动与轮盘均使用 macOS 系统热键注册；轮盘本身无需辅助功能权限，只有模拟按键、粘贴等对应操作按需请求权限。

### 界面与可靠性

- 悬浮面板的设置齿轮会显式激活并打开 Settings Scene；设置窗口遵循标准应用切换层级，并补齐可关闭、最小化、缩放的三色交通灯。
- 关闭轮盘全局模式后，编辑器会同步切换到默认应用预设；轮盘状态提示会显示用户当前设置的真实快捷键。

### 构建与分发

- 应用版本、构建号、Codex 客户端标识、安装说明和通用 DMG 文件名统一更新为 3.0.0（构建 300）。

## 2.6.0 — 2026-07-20

### 面板交互修复

- Fast、Plan、推理旋钮和语音操作改为读取当前 `~/.codex/keybindings.json`，不再因 Codex 快捷键调整而失效。
- 摇杆支持直接点击四个方向，并将左右切换统一到 Codex 的上一/下一任务快捷键。
- 面板语音键改为点击开始、再次点击结束；物理按键映射继续保持按住/松开语义。
- 摇杆和语音在激活 Codex 后改走系统 HID 事件通道，修复 Electron 未接收定向 PID 事件但面板仍显示“成功”的假反馈。
- 非激活悬浮面板触发动作时强制将 Codex 切到前台，并等待系统确认后再发送 HID 事件，避免事件误发给微信等原前台应用。
- 摇杆左右改为调用 Codex 的 `Previous Chat / Next Chat` 原生菜单；语音键避开 macOS 通用“开始听写”菜单，改为读取 Codex 的 `composer.startDictation` 配置并发送带完整修饰键按下/释放序列的快捷键。
- 摇杆左右不再依赖 Codex 的标签页导航栈，改为在面板 A1–A6 的真实任务 ID 列表中循环并通过 `codex://threads/<id>` 打开上一个或下一个任务。

### 按键映射可靠性

- 单键与组合键统一改用 CGEventTap 一级事件监听，不再向系统申请 Carbon 独占热键；即使组合已被 macOS 或其他应用注册，也不会再显示“已被独占，当前未生效”。
- 组合映射完整处理按下、自动重复和松开事件，并继续过滤 CodeXMicro++ 自身合成的键盘事件，避免动作重复和回环触发。
- 新实例启动时会关闭相同 Bundle ID 的旧实例，防止多个监听器重复执行同一映射。
- 自定义按键设置页统一显示单键与组合键的辅助功能权限及监听状态。

### 构建与分发

- 应用版本、构建号、Codex 客户端标识、安装说明和通用 DMG 文件名统一更新为 2.6.0（构建 260）。

## 2.5.0 — 2026-07-19

### 物理按键映射

- 新增一级优先的物理按键映射路径，键盘任意单键均可直接绑定，包括 Home、字母、空格、Esc 与左右 Shift / Control / Option / Command。
- 命中的映射按键会拦截按下、自动重复和抬起事件；修饰键被映射后，其原生修饰标记也会从后续键盘事件中剥离，真正接管原按键功能。
- 单独轻点修饰键可录入修饰键本身，按住修饰键再按其他键则继续录入组合快捷键。
- 一级物理映射优先于使用同一按键或修饰键的组合快捷键；被覆盖的组合会明确标记为未生效。
- 带修饰键的组合继续使用 Carbon 全局热键，并改为独占注册，避免多个应用同时触发；若组合已被独占则拒绝新设置。
- 语音触发键统一为按住开始、松开结束，独立按键与组合快捷键均保留按下 / 抬起语义。
- 新增「自定义按键」设置页，集中展示监听权限、触发优先级、监听方式和全部动作绑定。

### 可靠性与隐私

- 组合键注册失败时原子回滚，避免新旧绑定同时丢失；重复判断改用键码与修饰键，不受键盘布局显示文本影响。
- 过滤自动重复和 CodeXMicro++ 合成事件，事件 tap 超时停用后会自动恢复。
- 更新辅助功能权限说明：键盘事件仅在内存中与用户绑定比较，未命中内容不保存或上传。

### 构建与分发

- 应用版本、构建号、Codex 客户端标识、安装说明和 DMG 文件名统一更新为 2.5.0（构建 250）。
- 本地运行与通用 DMG 构建在 `swift build` 因 Xcode License 不可用时，会回退到 Xcode 内置 Swift 6 工具链。
