# 更新日志

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## 2.5.0 — 2026-07-19

### 物理按键映射

- 新增一级优先的物理按键映射路径，键盘任意单键均可直接绑定，包括 Home、字母、空格、Esc 与左右 Shift / Control / Option / Command。
- 命中的映射按键会拦截按下、自动重复和抬起事件；修饰键被映射后，其原生修饰标记也会从后续键盘事件中剥离，真正接管原按键功能。
- 单独轻点修饰键可录入修饰键本身，按住修饰键再按其他键则继续录入组合快捷键。
- 一级物理映射优先于使用同一按键或修饰键的组合快捷键；被覆盖的组合会明确标记为未生效。
- 带修饰键的组合继续使用 Carbon 全局热键，并改为独占注册，避免多个应用同时触发；若组合已被独占则拒绝新设置。
- 语音触发键统一为按住开始、松开结束，独立按键与组合快捷键均保留按下 / 抬起语义。
- 新增「自定义按键」设置页，集中展示监听权限、触发优先级、监听方式和全部动作绑定。

### 可靠性与隐私

- 组合键注册失败时原子回滚，避免新旧绑定同时丢失；重复判断改用键码与修饰键，不受键盘布局显示文本影响。
- 过滤自动重复和 CodeXMicro++ 合成事件，事件 tap 超时停用后会自动恢复。
- 更新辅助功能权限说明：键盘事件仅在内存中与用户绑定比较，未命中内容不保存或上传。

### 构建与分发

- 应用版本、构建号、Codex 客户端标识、安装说明和 DMG 文件名统一更新为 2.5.0（构建 250）。
- 本地运行与通用 DMG 构建在 `swift build` 因 Xcode License 不可用时，会回退到 Xcode 内置 Swift 6 工具链。

## 2.0.1 — 2026-07-19

- 将面板合成的快捷键和文本事件直接投递到当前 Codex 进程，修复新版 Codex 中 FAST 等操作无响应的问题。
- 安装包沿用稳定的代码签名要求，减少应用更新后重复申请辅助功能权限。
- 版本号与通用 DMG 安装包更新为 2.0.1。

## 2.0.0 — 2026-07-19

### 全局快捷键

- 为 A1–A6、四向摇杆、FAST、同意、拒绝、新任务、标注切换、语音、Codex 状态以及推理强度增减提供全局快捷键。
- 支持直接右键面板按键或旋钮半区录制、改绑和清除快捷键，并持久化用户配置。
- 增加快捷键冲突检测、重复绑定迁移、录制超时与取消反馈，避免合成按键事件反向触发面板动作。

### 交互与分发

- 快捷键录制时临时激活悬浮面板，完成后恢复原先的前台应用。
- 扩展面板四边拖动区域，更新侧边版本标识，并优化辅助功能授权提示行为。
- 未配置 Developer ID 时，DMG 改用稳定的本地开发签名，减少升级后重复申请辅助功能权限。
- 延续 1.1.3 的悬浮位置设置与关闭后重新打开修复。

## 1.1.0 — 2026-07-17

首个公开开源版本。

### macOS 虚拟控制面板

- 原生 SwiftUI 4 × 4 悬浮面板，支持移动、等比例缩放、跨桌面与始终置顶。
- 最近 6 个 Codex 任务的实时状态与一键跳转。
- 四向任务摇杆、四档推理旋钮、Fast、同意、拒绝、新任务和按住说话。
- CodeX 状态键显示本周剩余 Token 与累计消耗，约每 20 秒刷新。
- 可搜索的分类工具箱，覆盖任务、代码、Git、导航、界面与常用工作流。
- Force Touch 触觉强度、机械按键音与辅助功能权限设置。
